Installation

Three ways to
Get Started

Pick the method that fits your workflow. All options deliver the full framework — no feature differences.

Via npm

Best for projects with a module bundler like Vite, Webpack, or Parcel. Import in your entry file.

npm install kaple-css

Via CDN

One line in your HTML head. No setup, no tooling — works instantly in any browser.

<link rel="stylesheet" href= "cdn.jsdelivr.net/npm/kaple-css/dist/kaple.min.css"

Download CSS File

Download the compiled CSS file from GitHub and host it yourself for full offline control.

kaple.css — 52kb Download

About KapleCSS

KapleCSS is a minimal premium CSS framework designed to help developers create beautiful, consistent websites with ease. Built on a foundation of design tokens, it provides a single source of truth for colors, typography, and spacing — keeping every project cohesive from day one.
