Comenzar

Instalación

Tres formas de usar Kaple CSS. La más sencilla: un link en tu HTML.

A través de NPM (Recomendado para proyectos grandes)

NPM
# Instalar vía npm
npm install kaple-css

# En tu archivo principal de JS/TS o CSS
import 'kaple-css';

Vía CDN (Rápido para prototipos)

HTML
<!-- unpkg -->
<link rel="stylesheet" href="https://unpkg.com/kaple-css/dist/kaple.css">

<!-- o jsdelivr -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kaple-css/dist/kaple.css">

Archivo local

HTML
<!-- En tu <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
<link rel="stylesheet" href="kaple.css">

<!-- Sobrescribe la fuente -->
<style>
  :root { --font-sans: 'Inter', system-ui, sans-serif; }
</style>

Estructura mínima de página

HTML
<!DOCTYPE html>
<html lang="es" data-theme="light">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="kaple.css">
</head>
<body>
  <!-- Tu contenido aquí -->
</body>
</html>

Fundamentos

Design Tokens

Todos los valores del sistema están definidos como CSS Custom Properties en :root. Cambia un token y el sistema entero reacciona.

Cómo personalizar: Sobrescribe cualquier variable en tu propio CSS después de importar el framework. No necesitas modificar el archivo fuente.
CSS — Personalizar tokens
/* En tu CSS, después de importar kaple.css */
:root {
  /* Cambiar color de acento a azul */
  --color-accent:       #2563eb;
  --color-accent-hover: #1d4ed8;
  --color-accent-subtle:rgba(37, 99, 235, 0.08);

  /* Cambiar radios a más redondeado */
  --radius-sm: 8px;
  --radius-md: 12px;
  --radius-lg: 16px;
}
Token Valor por defecto Uso
--color-accent #000000 Color principal de acción (botones, links, focus)
--color-bg #ffffff Fondo base de la página
--color-text-primary #0a0a0a Texto principal
--color-border rgba(0,0,0,0.08) Bordes sutiles
--font-sans 'Geist', 'Inter', system-ui Fuente principal
--font-mono 'Fira Code', monospace Fuente para código
--radius-md 8px Radio de borde estándar
--shadow-md 0 4px 6px ... Sombra estándar
--transition-base 150ms ease Transición estándar

Scroll reveal — patrón con IntersectionObserver
// Revelar elementos al entrar en viewport
const observer = new IntersectionObserver((entries) => {
  entries.forEach(e => {
    if (e.isIntersecting) {
      e.target.style.opacity = '1';
      e.target.style.transform = 'translateY(0)';
      observer.unobserve(e.target);
    }
  });
}, { threshold: 0.1 });

document.querySelectorAll('.card, .feature-card').forEach(el => {
  el.style.opacity = '0';
  el.style.transform = 'translateY(12px)';
  el.style.transition = 'opacity 0.5s ease, transform 0.5s ease';
  observer.observe(el);
});

Dark Mode

Soporte nativo por sistema operativo (prefers-color-scheme) y toggle manual con data-theme.

Toggle manual con JS
// Toggle entre light y dark
function toggleTheme() {
  const html = document.documentElement;
  const isDark = html.getAttribute('data-theme') === 'dark';
  html.setAttribute('data-theme', isDark ? 'light' : 'dark');
  // Opcional: persistir en localStorage
  localStorage.setItem('theme', isDark ? 'light' : 'dark');
}

// Restaurar preferencia al cargar
const saved = localStorage.getItem('theme');
if (saved) document.documentElement.setAttribute('data-theme', saved);
Sobrescribir tokens en dark
/* Personalizar colores solo en dark mode */
[data-theme="dark"] {
  --color-accent:       #60a5fa; /* azul en dark */
  --color-accent-hover: #93c5fd;
}
