Documentación

Aprende a construir interfaces modernas y consistentes con el diseño de vanguardia.

Kaple CSS es un framework orientado a diseño premium. No es solo un conjunto de utilidades, sino un sistema cohesivo que prioriza la legibilidad, el contraste y la sensación de calidad espacial.

Instalación

NPM:

npm install kaple-css

CDN:

<link rel="stylesheet" href="https://unpkg.com/kaple-css/dist/kaple.css" />
